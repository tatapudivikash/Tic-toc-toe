Tic-Tac-Toe Game
Project Description
This is a simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript. It is a two-player game where players take turns placing X and O on a 3×3 grid. The game automatically detects a winner or a draw and allows players to restart the game.
Features
Two-player gameplay (Player X vs Player O)
Winner detection
Draw detection
Restart game button
Responsive and clean user interface
Built using only HTML, CSS, and JavaScript

How to Run the Project
Download or clone the project.
Open the project folder.
Double-click index.html or open it in any modern web browser.
Start playing the game.
How to Play
Player X starts the game.
Players take turns clicking on an empty cell.
The first player to align three symbols horizontally, vertically, or diagonally wins.
If all nine cells are filled and no player wins, the game ends in a draw.
Click the Restart button to play again.
Game Rules
Only one move is allowed per cell.
Players alternate turns automatically.
The game ends when a player wins or when all cells are filled.
No additional moves can be made after the game ends until it is restarted.
